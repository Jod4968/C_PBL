
📌 Leave Management System (C Project)
🧾 Project Description

This is a simple Leave Management System developed in the C programming language.
The system allows employees to apply for leave and enables an admin to approve or reject leave requests.
All data is stored using file handling (leave.txt).

🎯 Features
✅ Apply for leave
📄 View all leave records
✔️ Approve leave requests
❌ Reject leave requests
💾 Data stored using file handling
🛠️ Technologies Used
C Programming Language
File Handling (fopen, fprintf, fscanf)
Structures (struct)
